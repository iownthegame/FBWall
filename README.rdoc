== FBWall

FBWall is an example to fetch information from your Facebook wall and split them into different categories such as links, videos, photos, posts, etc., implemented by Ruby on Rails.


== Live Demo

You can find the example here
link:http://chickwall.herokuapp.com/

== Getting Started

1. Add an App on Facebook

2. Use Facebook SDK to get the login information

3. Fetch dats through Facebook API and split them into categories
